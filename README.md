## ✦ What is PitchCraft?

PitchCraft is a single-file, fully interactive proposal generation platform built for digital marketing agencies. Load a template, fill in a brief, and generate a complete, branded, 7-section client proposal — all in the browser with zero backend.

It was built to solve a real problem: agencies spending 3–5 hours writing proposals that should take 15 minutes.
